# DEEP DIVE — ミニゲーム仕様書

## 概要

縦穴ランダム生成プラットフォーマー型のダイビングゲーム。
プレイヤーは海底の縦穴を潜り、アイテムを拾い集め、酸素が尽きる前に地上へ帰還する。

- **実装**: React JSX（単一ファイル）。Canvas 2D描画。依存ライブラリなし（React hooks のみ使用）。
- **対応**: PC（キーボード）/ スマートフォン（バーチャルパッド）
- **想定プレイ時間**: 1ダイブ30秒〜2分。装備MAXまでの総プレイ時間は1〜3時間程度。

---

## コアループ

```
タイトル → ショップ → ダイブ → リザルト → ショップ → ...
```

1. **ショップ**: 所持金で装備を強化する
2. **ダイブ**: 縦穴に潜り、アイテムを拾い集める
3. **帰還判定**:
   - 地上に帰還成功 → **全アイテム持ち帰り**
   - 酸素切れ → **ランダムで1個だけ持ち帰り**（残り全ロスト）
4. **リザルト**: 持ち帰ったアイテムを売却 → 所持金に加算
5. ショップへ戻り繰り返す

## マップ構造

- **タイルベース**: 28px × 28px タイル。横14列 × 縦120行。
- **ランダム生成**: 毎回異なる地形。左右の壁がランダムに蛇行し、内部にランダムな足場が配置される。
- **深度ゾーン**: `depthZone = floor(row / 3)` で4段階に分かれる。
- **海底（row 117-119）**: 最下部2行分のオープンチャンバー + 最終行が全壁（床）。

### 深度ゾーン

| ゾーン名 | depthZone | 酸素消費/frame | 暗さ | 背景色 |
|----------|-----------|---------------|------|--------|
| 浅層 | 0-7 | 0.08 | 0 | `#1a6b8a` |
| 中層 | 8-19 | 0.15 | 0.3 | `#0d3f5e` |
| 深層 | 20-34 | 0.28 | 0.6 | `#061a2e` |
| 最深部 | 35+ | 0.40 | 0.85 | `#020a14` |

## プレイヤー物理

- **重力**: `vy += 0.4 * 0.4 = 0.16` /frame
- **水の抵抗**: `vy *= 0.92` /frame
- **終端下降速度**: 約 1.84 px/frame（自由落下時）
- **上昇（泳ぐボタン押下時）**: `vy = max(vy + swimForce * 0.15, swimForce)`
- **左右移動**: 即時に `vx = ±speed`。入力なし時 `vx *= 0.7`（減衰）
- **壁衝突**: タイルベースのAABB判定。X→Y の順で解決。

## アイテム

拾得はコストなし（触れるだけ）。ただし0.3秒の拾うアニメーション中はほぼ動けない。
被ダメージで拾い中断される。

### アイテムテーブル

| 名前 | 絵文字 | 出現depthZone | 売値(G) | 出現率 |
|------|--------|-------------|---------|--------|
| 貝殻 | 🐚 | 0-6 | 15 | 0.60 |
| ヒトデ | ⭐ | 0-8 | 25 | 0.50 |
| サンゴ片 | 🪸 | 3-10 | 50 | 0.40 |
| 真珠 | 🔮 | 8-18 | 150 | 0.30 |
| 古代コイン | 🪙 | 10-22 | 280 | 0.18 |
| 琥珀 | 💎 | 15-26 | 450 | 0.15 |
| 黒真珠 | 🖤 | 20-32 | 900 | 0.12 |
| 古代の像 | 🗿 | 25-40 | 1,600 | 0.08 |
| 深海の宝石 | 💠 | 35+ | 3,000 | 0.06 |
| 竜の涙 | 🐉 | 37+ | 6,000 | 0.03 |
| 海底の王冠 | 👑 | 海底固定 | 8,000 | 確定 |

- 各ゾーンではそのゾーンのdepthZone範囲内のアイテムのみ出現する。
- 浅層のアイテム（貝殻等）は深層では出ない（maxD制限）。
- **海底の王冠**は海底チャンバーに毎回1個確定配置。黄金のオーラ演出つき。

### 価値カーブ設計意図

深く潜るほど報酬が指数的に増加する。浅瀬周回では装備強化が遅く、深層に到達できるほど加速する。

## 障害物

すべて回避系（プレイヤーが敵を直接倒す手段はモリのみ）。

| 種別 | 挙動 | 接触時ダメージ |
|------|------|-------------|
| クラゲ 🪼 | 左右にサイン波で浮遊。depthZone 5以降に出現。深いほど密度が増す。 | 酸素 -8、プレイヤーをノックバック |
| ウニ 🦔 | 壁の上面に固定配置。動かない。 | 酸素 -5、上方にノックバック |
| 水流 →→→ | 横方向に一定の力を与えるエリア。depthZone 7以降。 | ダメージなし。移動を阻害/補助 |

---

## モリ（投擲武器）

- 有限リソース。装備「モリ」のレベルで所持本数が決まる。
- プレイヤーの向いている方向に直線発射。
- **クラゲに命中** → クラゲを消滅させる。
- **壁に命中** → 消滅。
- **ウニ・水流には効果なし**。
- クールダウン: 0.3秒。

---

## 装備強化

ショップでアイテム売却金を使って強化する。
コスト計算: `floor(baseCost × costMultiplier^currentLevel)`

| ID | 名称 | 効果 | MAX Lv | 初期コスト | コスト倍率 | 効果計算式 |
|----|------|------|--------|----------|----------|-----------|
| tank | タンク | 酸素最大量 | 15 | 50G | ×1.5 | `100 + Lv × 45` |
| fin | フィン | 移動速度 | 8 | 40G | ×1.7 | `2.8 + Lv × 0.4` |
| light | ライト | 視界半径(px) | 8 | 60G | ×1.8 | `80 + Lv × 30` |
| spear | モリ | 所持本数 | 6 | 70G | ×2.0 | `1 + Lv` |
| swim | 泳力 | 上昇力 | 8 | 80G | ×1.6 | `2.5 + Lv × 0.4` |

### 到達可能深度の目安

| 装備水準 | タンクLv | 泳力Lv | 安全往復可能な深度 |
|---------|---------|--------|-----------------|
| 初期 | 0 | 0 | ~46m（中層入口） |
| 序盤 | 4 | 2 | ~103m |
| 中盤 | 8 | 4 | ~148m |
| 深層攻め | 10 | 5 | ~166m |
| 王冠到達（約70%強化） | 11 | 6 | ~172m（海底） |
| MAX | 15 | 8 | 172m＋大幅余裕 |

## 演出・UX

### 酸素低下時の演出

- 酸素残量35%以下で画面端に赤いビネット（脈打つ）。
- 酸素ゲージが白く点滅。
- 深いほどビネットの強度が増す。

### アイテム拾得アニメーション

- アイテムに触れると約0.3秒（18フレーム）間プレイヤーが減速。
- アイテムの周囲にリング型プログレスが回る。
- 完了でアイテム取得＋パーティクル演出。
- 被ダメージで拾い中断。

### 新記録演出

- 5m単位で過去の最深記録を更新すると「🏆 最深記録更新」がフェードで表示。

### リザルト画面

- 帰還成功/酸素切れの表示。
- 持ち帰りアイテムを順番にフェードイン。
- 酸素切れ時: 失ったアイテムの総額を強調表示。最も高額なロストアイテムを大きく表示。
- 到達深度と新記録フラグを表示。

### HUD

- 左上: 酸素ゲージ（色変化: 青→橙→赤）、深度表示、ゾーン名、モリ残数、最深記録。
- 右上: 拾得アイテム数と合計金額。
- 右端: 地上までの距離ゲージ（縦バー + 三角マーカー + 距離テキスト）。

### 暗さ

- 深度ゾーンに応じて画面が暗くなる。
- プレイヤー中心に円形の光（ライト装備で半径が拡大）。

## 組み込み時の注意点

### ファイル構成

`diving-game.jsx` — Reactコンポーネント1ファイル。`export default` で `DeepDive` コンポーネントをエクスポート。

### 依存

- React (`useState`, `useEffect`, `useRef`, `useCallback`)
- それ以外の外部ライブラリなし

### マウント

```jsx
import DeepDive from './diving-game';

// 全画面で表示する場合
<DeepDive />

// 親要素のサイズに追従する（canvas は parentElement のサイズを使用）
<div style={{ width: 400, height: 600 }}>
  <DeepDive />
</div>
```

### セーブデータ

現在の実装にはセーブ機構がない。組み込み先で永続化したい場合、`gameRef.current` から以下を保存・復元する:

```javascript
{
  money: number,       // 所持金
  dives: number,       // ダイブ回数
  upg: {               // 装備レベル
    tank: number,
    fin: number,
    light: number,
    spear: number,
    swim: number
  },
  bestDepth: number    // 最深記録
}
```

### カスタマイズ可能な定数

| 定数 | デフォルト | 説明 |
|------|----------|------|
| `TILE` | 28 | タイルサイズ(px) |
| `COLS` | 14 | マップ横幅(タイル数) |
| `rows`(genMap内) | 120 | マップ縦幅(タイル数) |
| `ZONES[].rate` | 0.08-0.40 | 各ゾーンの酸素消費/frame |
| `ITEMS_DB[].val` | 15-8000 | アイテムの売却価格 |
| `UPG[].bc/cm/mx/fn` | 各種 | 装備のコスト・効果 |

### 既知の制限事項

- サウンドなし（組み込み先で Web Audio API 等を追加可能）
- セーブなし（上記参照）
- 画面サイズは親要素に追従するが、極端に横長/縦長な画面ではパッドの位置が最適でない場合がある
- `requestAnimationFrame` ベースのため、フレームレートは端末依存（酸素消費等はフレーム単位で計算される）
